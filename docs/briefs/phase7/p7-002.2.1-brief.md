# P7-002.2.1 定义 external source allowlist

- Wave: 7C Usage Fallback Ladder
- Group: 外部升级
- Priority: P1
- Value: 中
- Depends on: P7-002.2
- Type: spec

## Context

external fallback 必须有可信边界，否则宿主会把低可信来源和官方来源混在一起。

## Acceptance Criteria

- **Artifact**: `docs/specs/2026-04-23-p7-002-usage-fallback-ladder-v1.md`
- **Command**: `rg -n "external source allowlist|官方文档|官方 GitHub|可信第三方" docs/specs/2026-04-23-p7-002-usage-fallback-ladder-v1.md`
- **Assertion**: 文档限定 external allowlist 为官方与可信来源，并明确不鼓励的低可信来源。
- **Output**: `external allowlist 固定。`

## Steps

1. 列出允许的官方/可信外部来源
2. 说明不鼓励的非可信来源
3. 写清这条边界用于控制 fallback 质量
